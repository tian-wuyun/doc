INTRODUCTION                                  *vim-gfm-syntax-introduction*

*gfm-syntax* 为 GitHub Flavored Markdown 提供额外的语法突出显示。GitHub 将他们的额外功能添加到 Markdown。这个插件填补了普通 Markdown 和 GitHub Flavored Markdown 之间的空白

Ref: https://help.github.com/categories/writing-on-github/

此插件在“markdown”文件类型上自动启用。如需更多自定义，请参阅自述文件

https://github.com/rhysd/vim-gfm-syntax#readme


==============================================================================
INSTALL                                            *vim-gfm-syntax-install*

如果您不使用任何插件管理器，请将 after/、plugin/ 和 doc/ 中的所有文件复制到 .vim 目录

如果您使用插件管理器，请按照它的说明安装 |gfm-syntax|.

下面是|neobundle.vim| 的例子.
>
    NeoBundle 'rhysd/vim-gfm-syntax'
<
